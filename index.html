demo展示

info弹窗

var dialog = new Dialog({
    type: "info",
    infoType: "success", //error/warning
    content: "删除成功！",
    autoClose:2000,
    overlayClose: true
});

info弹窗一般分为成功，失败，警告弹窗
infoType:指定提示类型success为成功
conent:包含弹出层的内容
autoClose:定时关闭,在info中不传默认为2000
overlayClose:开启点击遮罩关闭弹窗

成功弹窗 失败弹窗 警告弹窗

confirm弹窗

var btnOk = {
	isShow:true,
	text:"点击我不消失",
	callback:function(){
		return false;
	}
}
var btnCancel = {
	isShow:true,
	text:"取消",
	callback:function(){
		return true;
	}
}
var dialog = new Dialog({
    type: "confirm",
    content: "你确定要删除选中列？",
    btnOk:btnOk,
    btnCancel:btnCancel
});

confirm弹窗一般不用指定infoType,默认为warning
confirm弹窗默认配置一个标题，和两个按钮，如果没有设置title值，那么默认显示“提示”，如果不需要显示标题，那么title传空
content必须要有内容，如果两个要配置两个按钮不显示，那么在btnOk和btnCancel的isShow传false即可
btnOk : 确定按钮的对象，回调函数callback返回false将不关闭弹出层。
btnCancel : 取消按钮的对象，回调函数callback返回false将不关闭弹出层
如果没有给按钮传回调函数,那么默认返回true。

运行 两个弹窗 
var btnOk = {
	callback:function(){
		return false;
	}
}
var btnCancel = {
	isShow:true,
	text:"取消",
	callback:function(){
		return true;
	}
}
var dialog = new Dialog({
    type: "confirm",
    width:480,
    content: "你确定要删除选中列你确定要删除选中列？",
    btnOk:btnOk,
    btnCancel:btnCancel
});

当内容较长时，建议传width值480

window弹窗

var dom = '<div>一个文档，dom文件，或者iframe。</div>';
var dialog = new Dialog({
    type: "window",
    content: dom,
    width:1000,
    height:500,
    btnOk:{
    	isShow:true,
    	text:"自定义名称",
    	callback:function(){
    		return true;
    	}
    }
});

window弹窗一般不用指定infoType
一般引入一个文档，一段代码，一个页面时使用
一般要自行设置高宽 width : 自定义弹出框的宽度，只需要传数字，不要带单位 height : 自定义弹出框的高度，只需要传数字，不要带单位

运行

window弹窗点取消和关闭按钮不关闭

var dom = '<div>一个文档，dom文件，或者iframe。</div>';
var dialog = new Dialog({
    type: "window",
    content: dom,
    width:1000,
    height:500,
    btnOk:{
    	isShow:true,
    	text:"自定义名称",
    	callback:function(){
    		return true;
    	}
    },
    btnCancel:{
    	isShow:true,
    	text:"不关闭",
    	callback:function(){
    		return false;
    	}
    },
    btnClose:{
    	isShow:true,
    	text:"",
    	callback:function(){
    		return false;
    	}
    }
});

window弹窗一般不用指定infoType
一般引入一个文档，一段代码，一个页面时使用
一般要自行设置高宽 width : 自定义弹出框的宽度，只需要传数字，不要带单位 height : 自定义弹出框的高度，只需要传数字，不要带单位

参数列表

参数 参数类型 默认参数 参数说明
id String 随机数

弹出窗口唯一标示符

用于区分多个弹出窗口
type String null

弹出窗口模板

模板的参数：

info: 消息提示（默认没有任何按钮，没有标题，没有关闭按钮，建议传autoClose设定定时关闭，传overlayClose添加点击遮罩层关闭弹框功能）

confirm: 确认弹框（默认有提示的标题，带有确定和取消按钮）

window: window内容弹框 （默认带有确定按钮）
[infoType] String null

弹出窗口提示类型

提示类型的参数：

success: 成功

error: 失败或者错误

warning: 警告
[title] String null

弹出窗口标题

可以定义标题,默认标题为“提示”
content String

弹出窗口内容

参数可以是字符串，也可以是HTML代码。
[width] Number 320

弹出窗口宽度
[height] Number auto

弹出窗口高度
[offsetX] Number 自动计算

弹出窗口左偏移量
[offsetY] Number 自动计算

弹出窗口上偏移量
[isOverlay] Boolean true

是否显示遮罩

默认为显示
[overlayClose] Boolean false

是否开启点击遮罩层关闭弹框的功能

默认为不开启
[autoClose] Boolean false

弹出窗口自动关闭时间

以毫秒为单位，只能传数字，一般在type传值为“info”时使用
[btnOk] Object null

确定按钮操作

object{

isShow:true, //是否显示确认按钮

text:"确定", //确定按钮的文字

callback:function(){} //确定之后回调函数

}
[btnCancel] Object null

取消按钮操作

object{

isShow:true, //是否显示取消按钮

text:"确定", //取消按钮的文字

callback:function(){} //取消之后回调函数

}
[btnClose] Object null

关闭按钮操作

object{

isShow:true, //是否显示关闭按钮

callback:function(){} //关闭之后回调函数

}
[autoCloseFn] function null

定时关闭弹窗之后回调函数 (新增2016-03-03)
